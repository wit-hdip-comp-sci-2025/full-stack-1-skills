## When to use

Use this skill when you need to:

- Style a Hapi/Handlebars app with a CSS framework
- Build responsive layouts, forms, and navigation
- Apply consistent components (navbar, buttons, notifications, tables)

## How to use

- **Bulma CSS**: [rules/bulma-css.md](rules/bulma-css.md) — Bulma setup, layout structure, navbar, forms, components, typography.

Refer to the rule that matches your choice of framework; more options can be added to `rules/` later.
